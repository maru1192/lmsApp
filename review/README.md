# ①課題名
週次学習の振り返りフォーム（学習の記録アプリ）

## ②課題内容（どんな作品か）
- 学習の振り返りを毎週同じフォーマットで記録できるWebフォーム
- 達成率（スライダー）、達成要因/未達要因（自由記述）、感情（複数選択）、感想（自由記述）に加え、来週の学習時間目標（曜日別入力＋合計）、来週のカリキュラム目標（プルダウン）、来週のオリジナル目標・相談事項を入力して送信。
- 送信内容は `write.php` で受け取り、`data/data.txt` に追記保存、同時に「記録完了ページ」で入力内容を見やすく整形して表示

## ③アプリのデプロイURL
http://olive5g72.sakura.ne.jp/1226_questionnaire/

## ⑤工夫した点・こだわった点
- テキストだけで回答させるフォームではなく、スライダーやチェックボックスなども活用しながら、感覚的に回答しやすいフォームを意識して作成しました。
- 感情を選択する部分では、丸いアイコンをクリックするだけで選択できる形にしました。
- 日本時間時刻で保存されるようにしました。  
- 感情（`emotions[]`）が配列でPOSTされる（そのまま表示するとarryになる）ため、`implode()` で文字列化して保存・表示できるようにしました。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 保存した情報の一覧表示
- 過去のデータを踏まえた集計データが表示されるページの作成（AI系のAPIとも絡ませたい）
- メンター側がコメントを入れられる部分を作成
